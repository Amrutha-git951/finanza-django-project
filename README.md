# Finanza Django Website

This is a Django project built using the Finanza template.

1. 📝 Description
This website includes:
- Home page
- About page
- Contact page
- Project page
- Testimonial page
- My CV page

It uses Django templates, static files (CSS, JS, images), and Bootstrap for styling.

2. ## 💻 Project Structure

 myproject/

├─ core/                 # Your Django app
├─ templates/            # HTML templates
├─ static/               # CSS, JS, images
├─ myproject/            # Django project settings
└─ db.sqlite3            # Database file
